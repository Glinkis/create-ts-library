# create-ts-library

## File Structure

```
project
├─ src
└─ test
```

## CLI

`create-ts-library <flags>`

| Flag          | Description                 |
| ------------- | --------------------------- |
| `<no flag>`   | Run a full build.           |
| `-d, --dev`   | Build a development bundle. |
| `-p, --prod`  | Build a production bundle.  |
| `-t, --test`  | Run tests.                  |
| `-w, --watch` | Run in watch mode.          |
| `-h, --help`  | Display commands.           |
